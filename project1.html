My Favorite Old Game

Fallout 1 was released in 1997 by Interplay. The game takes place in post apocalyptic america mainly taking place in Southern California

Why I like the game so much

Fallout 1 is a very fun RPG. It is set in the future so you get to use laser weapons and power armour. The game is full of dark humour that is very entertaining. So in short the reasons I like the game are

  1. Its a RPG
  2. Good Humour
  3. Laser Guns

Reasons you should try this game

  • Fun RPG
  • It is considered a Classic
  • It is old so it is very cheap
